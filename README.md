# Data-Science-College-Application-Analysis

A Data Science and Machine Learning project designed to analyze the primary factors that determine college acceptance and to predict if a student will get into a college of their choice.

## Essential Questions

- What are the primary factors that affect college acceptance?
- Can we predict if a student will get into a certain college or not?

## Technologies Used

- R 4.6.0
- RStudio
- Jupyter Lab

## Features

- Acceptance Rate of the dataset
- Random Forest Model (used for prediction)
- Logistic Regression Model (used for prediction)
- Example predictions for both models
- Comparison of the two models
- Impact of factors that affect college acceptance
- Relationship between individual factors and acceptance
- Correlation between individual factors and acceptance

## Skills Gained

- Using R to analyze data and create visualizations
- Using R to clean data 
- Using markdown and R in ipynb files
- Using Jupyter Lab and RStudio
- Using the Random Forest model and the Logistic Regression model to predict outcomes
